# 前端移动应用技能

## 核心技术栈

### React Native
- **类型**：移动应用框架
- **描述**：使用JavaScript和React构建原生移动应用的框架
- **应用**：整个移动应用的UI构建，包括登录界面、聊天界面、好友列表等

### Expo
- **版本**：~51.0.39
- **类型**：React Native开发平台
- **描述**：提供了一套工具和服务，简化React Native应用的开发和部署
- **应用**：项目初始化、依赖管理、构建和发布

### Socket.io-client
- **版本**：^4.8.1
- **类型**：实时通信
- **描述**：Socket.IO的客户端库，用于与服务器建立WebSocket连接
- **应用**：实现实时消息接收、用户在线状态同步

### TypeScript
- **类型**：编程语言
- **描述**：JavaScript的超集，添加了静态类型检查
- **应用**：提高代码可维护性和可靠性，减少运行时错误

### React Navigation
- **类型**：导航库
- **描述**：React Native应用的导航解决方案
- **应用**：实现应用内页面跳转、导航栏配置

## 项目结构

```
mobile/
├── assets/              # 静态资源
│   ├── adaptive-icon.png # 自适应图标
│   ├── favicon.png      # 网站图标
│   ├── icon.png         # 应用图标
│   └── splash.png       # 启动屏幕
├── src/                 # 源代码
│   ├── api.ts           # API调用封装
├── .env.example         # 环境变量示例
├── App.tsx              # 应用根组件
├── README.md            # 项目说明
├── app.json             # Expo配置
├── babel.config.js      # Babel配置
├── package.json         # 项目配置
└── tsconfig.json        # TypeScript配置
```
