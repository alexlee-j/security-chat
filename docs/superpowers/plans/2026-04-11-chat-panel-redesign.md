# 聊天面板重构计划

## 目标
根据 Figma 设计稿重构聊天主界面，同时支持 Light/Dark Mode。

## 设计规范（来自 Figma）

### 整体布局
- 侧边栏：280 x 700
- 聊天区域：920 x 700

### 侧边栏设计（参考已完成的抽屉导航）
- 搜索框：圆角 16px，背景 #edf0f2 (dark: #2c2c2e)
- 会话卡片：
  - 圆角 12px，高度 72px
  - 选中态：蓝色背景 (#3390ec)，白色文字
  - 未选中：白色背景 (#ffffff)，hover 灰色背景
  - 头像 40x40，圆形
  - 名字 + 时间在同一行
  - 消息预览在下方
  - 未读徽章：圆形白底蓝字

### 聊天区域设计
**Header（64px）：**
- 用户头像：40x40，圆形，蓝色 (#3390ec)
- 名字 + 在线状态
- 右侧操作按钮：🔍 📞 📹 ⋮

**消息区域：**
- 背景：#e8f4f8 (dark: #1a1a1a)
- 接收消息气泡：白色 (#ffffff)，左侧
- 发送消息气泡：浅绿色 (#e5ffde)，右侧，带 ✓✓ 已读标记

**输入区域（77px）：**
- 白色背景 (#ffffff)，顶部边框 #dfe1e5
- 输入框：灰色背景 (#f8f9fa)，圆角 22px
- 按钮：发送、文件、表情、语音

## 需要备份的文件
1. `apps/desktop/src/features/chat/chat-panel.tsx`
2. `apps/desktop/src/features/chat/conversation-sidebar.tsx`
3. `apps/desktop/src/styles.css`（相关样式部分）
