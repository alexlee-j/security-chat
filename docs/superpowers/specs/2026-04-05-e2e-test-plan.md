## 7. Tester A 测试命令脚本

```bash
#!/bin/bash
# Tester A E2E 测试脚本

# 1. 验证后端
curl http://localhost:3000/api/v1/health

# 2. 注册用户
REG=$(curl -X POST http://localhost:3000/api/v1/auth/register ...)
TOKEN=$(echo $REG | grep -o '"accessToken":"[^"]*"' | cut -d'"' -f4)

# 3. 登录
LOGIN=$(curl -X POST http://localhost:3000/api/v1/auth/login ...)
TOKEN=$(echo $LOGIN | grep -o '"accessToken":"[^"]*"' | cut -d'"' -f4)

# 4. 创建会话
CONV=$(curl -X POST http://localhost:3000/api/v1/conversation/direct \
  -H "Authorization: Bearer $TOKEN" ...)

# 5. 发送消息
MSG=$(curl -X POST http://localhost:3000/api/v1/message/send \
  -H "Authorization: Bearer $TOKEN" ...)
```
