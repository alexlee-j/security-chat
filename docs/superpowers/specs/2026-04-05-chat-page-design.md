# Security Chat - 聊天页面设计方案

## 1. 概述

本文档定义 Security Chat 桌面端应用页面的视觉设计方案，包含登录、注册和聊天页面，支持 Light/Dark 主题切换。

**设计参考**: Telegram Desktop 设计语言
**主题切换**: 用户可在设置页面主动切换，切换时有平滑过渡动画

## 3. 登录与注册页面

### 3.1 页面布局（通用）

登录与注册页面采用桌面端原生分屏布局：

```
┌──────────────────────────────────────────────────────┐
│                                                      │
│  品牌区 (540px)          │  表单区 (660px)           │
│  渐变背景 + 装饰圆形      │  表单卡片居中              │
│  Logo + 标题 + 标语      │  输入框 + 按钮 + 链接     │
│                                                      │
└──────────────────────────────────────────────────────┘
```

**布局规格**:
- 页面总宽度: 1200px
- 页面总高度: 700px
- 品牌区: 540px 宽（45%）
- 表单区: 660px 宽（55%）
- 表单卡片: 420px 宽，居中显示

### 3.2 品牌区详细结构

```

│    ○ ← 装饰圆形（3层）       │

│    安全至上，畅聊无忧         │  ← 标语 19px 主题色

└──────────────────────────────┘
```

**品牌区规格（Light Mode）**:
- 背景: 渐变 #e8f4f8 → #d4e4f8 + #3390ec 遮罩 8% opacity
- 装饰圆形: 3个 #3390ec 圆形，opacity 分别为 0.15 / 0.08 / 0.12，尺寸分别为 280px / 220px / 160px，位置分别为 (-60,-80) / (280,340) / (380,-20)

- 标题: "Security Chat" 32px 粗体，白色

**品牌区规格（Dark Mode）**:

- 装饰圆形: 3个 #8777d1 圆形，opacity 分别为 0.15 / 0.08 / 0.12，尺寸分别为 280px / 220px / 160px，位置分别为 (-60,-80) / (280,340) / (380,-20)

## 6. 主题切换

### 6.1 切换位置

**设置页面** - 提供主题选择器：
- Light Mode
- Dark Mode

### 6.2 设置抽屉

从右侧滑出的抽屉式侧边栏，包含：
- 主题选择器（Light/Dark 单选）
- 当前账号信息
- 退出登录按钮

### 6.3 过渡动画

切换时使用 CSS transition 实现平滑过渡：

```css
:root {
  transition: background-color 200ms ease, color 200ms ease;
}

* {
  transition: background-color 200ms ease, border-color 200ms ease;
}
```

### 6.4 主题持久化

- 使用 localStorage 存储主题偏好
- 页面加载时读取并应用主题

### 10.2 群聊页面规格

**整体布局**:
- 页面总宽度: 1200px
- 页面总高度: 700px
- 侧边栏: 宽度 280px，背景 #f8f9fa
- 聊天区域: 宽度 920px，背景 #e8f4f8

**顶部栏 (TopBar)**:
- 高度: 64px
- 背景: #ffffff
- 底部边框: 1px #dfe1e5
- 群头像: 48x48px 圆形，#3390ec 填充，白色"开"字
- 群名称: 15px 粗体，#000000
- 成员数: 12px，#707579
- 操作按钮: 视频通话 / 语音通话 / 更多，24x24px，#707579

**消息区域 (MessageArea)**:
- 背景: #e8f4f8（继承自聊天区域）
- 内边距: 20px
- 消息间距: 10px

**群聊消息气泡规格**:
- 发送者名称: 12px 粗体，颜色对应头像色
  - 李明: #3390ec（蓝色）
  - 张伟: #8777d1（紫色）
  - 王芳: #f09378（橙色）
- 气泡最大宽度: 320px
- 圆角: 18px，发送消息右下角 10px
- 气泡内边距: 10px 14px 6px 14px
- 文字: 14px，#000000
- 时间戳: 11px，#707579

**输入区域 (InputArea)**:
- 高度: 72px
- 背景: #ffffff
- 顶部边框: 1px #dfe1e5
- 输入框: 圆角 22px，背景 #f8f9fa，高度 44px
- 附件/表情按钮: 28x28px，#707579
- 发送按钮: 28x28px，#3390ec

**会话列表 (ChatList) — 群聊版本**:
- 项目1: 开发群，蓝色背景 #3390ec，白色文字
- 项目2: 产品讨论组，白色背景，阴影
- 项目3: 日常交流，白色背景，阴影
- 每个项目高度: 72px，包含48px头像 + 群名 + 最新消息 + 时间

**FAB 悬浮菜单**:
- FAB 按钮: 56x56px 圆形，#3390ec 背景，白色"+"图标，阴影
- 位置: 侧边栏右下角 (x:224, y:644)
- 悬浮菜单: 200x88px，圆角 12px，阴影，8px 内边距
  - 新建群聊: group_add 图标 + "新建群聊"文字
  - 添加好友: person_add 图标 + "添加好友"文字
