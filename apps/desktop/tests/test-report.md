## 修复的问题

### P0 - CryptoKey 序列化
使用 JWK 格式导出/导入 CryptoKey，解决 localStorage 存储问题。

### P0 - 消息加密/解密失败
修复密钥派生逻辑，使用 KDF 替代随机生成，确保双方密钥同步。

### P1 - 创建会话需要 UUID
支持用户名搜索，自动检测输入格式并搜索用户。

### P1 - 消息显示问题
添加跳过自己发送消息的解密逻辑。
